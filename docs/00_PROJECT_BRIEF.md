# 00. Project Brief

## 프로젝트명

잇다(:connect)

## 캐치프레이즈

아이와 도시를 잇다. 서울의 미래를 잇다.

## 프로젝트 정의

서울 25개 구의 아이동반 친화성을 250m 격자 기반 `유모차 생활보행 점수`로 시각화하는 GitHub Pages 기반 정적 웹 대시보드다.

이 프로젝트는 공개데이터를 활용해 의료, 행정, 교육, 여가 시설에 대한 생활권 접근성을 비교한다. 점수는 실제 현장 통행 가능성을 확정하지 않고, 생활시설 접근성을 상대적으로 이해하기 위한 시각화 지표로 사용한다.

## 핵심 질문

서울의 각 구는 아이와 함께 유모차로 의료, 행정, 교육, 여가 시설에 접근하기 쉬운가?

## 분석 대상

- 공간 범위: 서울특별시 25개 자치구
- 분석 단위: 250m x 250m 격자
- 출발점: 각 격자의 중심점
- 평가 시설: 의료, 행정, 교육, 여가 관련 생활시설
- 주요 결과: 격자별 점수, 구별 평균 점수, 카테고리별 점수

## 제외 대상

다음 주제는 프로젝트 범위에서 제외한다.

- 유모차/아이동반 이동성과 직접 관련 없는 다른 접근성 주제
- 실제 통행 가능 여부 확정
- 보도 폭, 단차, 공사, 불법주정차 등 현장 조사를 필요로 하는 판정

## 결과물

- GitHub Pages 기반 웹 대시보드
- 서울 25개 구 유모차 생활보행 점수 지도
- 250m 격자별 점수 heatmap
- 의료/행정/교육/여가 카테고리별 상세 보기
- 구별 순위
- 방법론, 데이터 출처, 한계 설명 페이지

## 핵심 산출물

- `public/data/seoul_districts.geojson`
- `public/data/grid_scores.geojson`
- `public/data/district_scores.json`
- `public/data/facilities.geojson`
- `public/data/category_summary.json`
- `public/data/metadata.json`
- 전처리 스크립트
- GitHub Pages 배포 설정
- 방법론 및 제출용 문서

## 표현상 주의

사용 가능한 표현:

- 유모차 생활보행 점수
- 아이동반 친화점수
- 공개데이터 기반 생활시설 접근성
- 아이와 함께 이동하기 좋은 생활권
- 접근성이 낮게 나타남
- 개선 여지가 있음

사용 금지 표현:

- 통행 가능 확정
- 통행 불가 확정
- 아이 키우기 나쁜 구
- 위험한 구
- 프로젝트 범위를 벗어난 다른 접근성 주제명

필수 한계 문구:

> 본 점수는 실제 유모차 통행 가능 여부를 확정하는 지표가 아니라, 공개데이터 기반 생활시설 접근성을 비교하기 위한 시각화 지표입니다. 실제 현장 여건은 보도 폭, 단차, 공사, 불법주정차, 시설 운영시간 등에 따라 달라질 수 있습니다.
