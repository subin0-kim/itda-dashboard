# 01. Product Requirements

## 제품 유형

`잇다(:connect)`는 서울 25개 구의 유모차 생활보행 점수를 탐색하는 정적 웹 대시보드다.

웹은 사전에 전처리된 GeoJSON, JSON, CSV 파일을 읽어 지도와 차트로 표현한다. 웹 브라우저에서는 거리 계산, 점수 계산, 데이터 생성 작업을 수행하지 않는다.

## 배포 방식

- GitHub Pages로 정적 배포
- GitHub Actions로 빌드 및 배포 자동화
- 백엔드 서버와 데이터베이스 없이 동작
- 모든 공개 데이터 산출물은 `public/data/` 아래에 배치

## 주요 사용자

- 서울시 빅데이터 활용 경진대회 심사위원
- 정책 및 도시 데이터 시각화에 관심 있는 사용자
- 아이동반 이동성과 생활시설 접근성을 비교하려는 시민
- 서울 자치구별 생활권 격차를 탐색하려는 기획자

## 사용자 목표

- 서울 25개 구의 유모차 생활보행 점수를 한눈에 비교한다.
- 특정 구의 의료, 행정, 교육, 여가 접근성 강점과 약점을 확인한다.
- 250m 격자 단위로 점수가 낮게 나타나는 생활권을 탐색한다.
- 하위 구가 참고할 수 있는 유사 조건 상위 구를 확인한다.
- 점수 산식, 데이터 출처, 한계를 이해한다.

## 핵심 기능

- 서울 25개 구 점수 지도
- 구별 TOP 5 / BOTTOM 5 순위
- 의료/행정/교육/여가 카테고리 필터
- 구 클릭 후 상세 페이지 이동
- 선택 구의 250m 격자 heatmap
- 주요 시설 위치 표시
- 카테고리별 시설 유형 필터
- 카테고리별 구 순위
- 벤치마킹 추천
- 방법론, 데이터 출처, 한계 설명
- 데이터 파일 누락 시 안전한 빈 상태 표시

## 비목표

- 백엔드 API 제공
- 사용자 계정, 로그인, 저장 기능
- 실시간 데이터 갱신
- 브라우저 내 거리 계산 또는 점수 계산
- 샘플 데이터, 더미 데이터, 임의 점수 생성
- 실제 통행 가능 여부 판정
- 현장 조사 기반 판정
- 유모차/아이동반 이동성과 직접 관련 없는 다른 접근성 분석
